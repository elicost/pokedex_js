# Pokedex (JavaScript Web App)
This app was built as a project for Achievement 1 of CareerFoundry's JavaScript Immersion course

A responsive, client-side Pokedex application built with vanilla JavaScript, jQuery, and Bootstrap 4.
The app uses the public PokeAPI to fetch the first 150 Pokemon and allow end users to browse, search, and view details for individual Pokemon.

## Features
    - Displays Pokemon in button grid
    - Fetches live data from API
    - Allows case-insensitive, partial-match search for Pokemon by name, with search function integrated into nav bar
    - Select from search to open details
    - Responsive layouts and popups built using Bootstrap 4

## Technologies Used

    - HTML5
    - CSS3
    - JavaScript (ES6+)
    - Bootstrap 4.3
    - jQuery (for Bootstrap support)
    - Fetch API
    - PokeAPI (https://pokeapi.co)

## Installation and Setup

This project runs entirely in the browswer, with no additional external applications necessary for the end user.
